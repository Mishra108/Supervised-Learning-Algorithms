# Supervised-Learning-Algorithms
 This project utilizes the Titanic dataset to study and apply various supervised learning algorithms for predicting passenger survival. The primary objective is to understand how different models perform on the same dataset and to evaluate their predictive accuracy.

The project focuses on key concepts such as data preprocessing, feature selection, and model evaluation. By implementing multiple algorithms, it provides a comparative understanding of their effectiveness in solving a classification problem.

Overall, this work demonstrates the practical application of supervised learning techniques and highlights the importance of selecting appropriate models for accurate predictions.

# Algorithms Used
Logistic Regression
KNN (K Nearest Neighbors)
Naive Bayes
Decision Tree
SVM (Support Vector Machine)
